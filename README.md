## Installation

### vim

dein.vim

```vim
call dein#add('ryota2357/necodark', #{ rev: 'vim' })
```

dein.vim (toml)

```toml
[[plugins]]
repo = 'ryota2357/necodark'
rev = 'vim'
```

vim-plug

```vim
Plug 'ryota2357/necodark', { 'branch': 'vim' }
```

dein.vim

```vim
call dein#add('ryota2357/necodark', #{ rev: 'nvim' })
```

dein.vim (toml)

```toml
[[plugins]]
repo = 'ryota2357/necodark'

```

packer.nvim

